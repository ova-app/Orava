# Choisir le Facteur d'Activité Physique (NAF)

Le facteur d'activité (ou NAF - Niveau d'Activité Physique) est un coefficient qui permet d'ajuster votre métabolisme de base (BMR) selon votre niveau d'activité physique. Il est utilisé pour calculer le **TDEE** (Total Daily Energy Expenditure), soit la dépense énergétique journalière.

## 📌 Valeurs standards :

| NAF  | Description                                          | Coefficient |
|------|------------------------------------------------------|-------------|
| 1.2  | **Sédentaire** : peu ou pas d'exercice               | 1.2         |
| 1.375| **Léger** : activité légère 1-3 jours/semaine        | 1.375       |
| 1.55 | **Modéré** : activité modérée 3-5 jours/semaine      | 1.55        |
| 1.725| **Intense** : activité intense 6-7 jours/semaine     | 1.725       |
| 1.9  | **Très intense** : travail physique + sport quotidien| 1.9         |

## 💡 Comment choisir ?

1. **Travail de bureau, peu de marche, pas de sport ?**
   → Choisissez **1.2**

2. **Vous faites du sport léger (marche rapide, vélo) quelques fois par semaine ?**
   → Choisissez **1.375**

3. **Vous êtes actif régulièrement, entraînement 3-5 fois/semaine ?**
   → Choisissez **1.55**

4. **Sport quotidien ou activité physique soutenue ?**
   → Choisissez **1.725**

5. **Travail physique + sport quotidien voire biquotidien ?**
   → Choisissez **1.9**

---

👉 **Conseil** : mieux vaut **sous-estimer** légèrement votre niveau d’activité si vous hésitez, surtout en cas de perte de poids.
